# Student Grade Calculator

This is a Java program that calculates student average marks and assigns grades based on the input marks.

##🔍 Features

- Accepts student marks
- Calculates average marks
- Displays grade based on average
- Simple user input

## Technologies Used

- Java
- Arrays
- Scanner Class

## How to Run

1. Compile the program
2. Run the program
3. Enter number of subjects
4. Enter marks
5. View average and grade

## Example

Input:
Enter number of subjects: 3  
Enter marks: 80 70 90  

Output:
Average: 80  
Grade: A
